# เชื่อม Store PP26 กับ Google Sheet (ฟรี)

แอปเก็บข้อมูลในเบราว์เซอร์ (localStorage) อยู่แล้ว รีเฟรชแล้วข้อมูลไม่หาย
ถ้าต้องการให้ข้อมูล **แชร์ข้ามเครื่อง / หลายคนเห็นตรงกัน** ให้เชื่อมกับ Google Sheet
ผ่าน Google Apps Script Web App ซึ่งใช้ได้ฟรี ไม่ต้องมีเซิร์ฟเวอร์

## ขั้นตอนตั้งค่า (ครั้งเดียว)

1. เปิด [Google Sheets](https://sheets.google.com) แล้วสร้างไฟล์ใหม่ 1 ไฟล์
2. เมนู **Extensions → Apps Script**
3. ลบโค้ดเดิมทั้งหมด แล้ววางเนื้อหาจากไฟล์ [`Code.gs`](./Code.gs) ลงไป
4. กด **Deploy → New deployment**
   - Select type: **Web app**
   - Execute as: **Me**
   - Who has access: **Anyone** (เพื่อให้แอปเรียกได้โดยไม่ต้องล็อกอิน)
5. กด **Deploy** แล้วอนุญาตสิทธิ์ (authorize) ตามที่ระบบถาม
6. คัดลอก **Web app URL** (ลงท้ายด้วย `/exec`)

## เชื่อมในแอป

1. เปิดแอป Store PP26
2. กดปุ่ม **⚙️ เชื่อม Google Sheet** ที่แถบเมนูซ้าย
3. วาง Web app URL ลงในช่อง แล้วกด **บันทึกลิงก์**
4. ใช้ปุ่ม
   - **ส่งข้อมูลขึ้น Sheet** — อัปโหลดข้อมูลปัจจุบันไปเก็บบน Google Sheet
   - **ดึงข้อมูลจาก Sheet** — โหลดข้อมูลล่าสุดจาก Sheet มาแทนที่ในเครื่องนี้

> ทุกครั้งที่ต้องการซิงก์ ให้กด "ส่งข้อมูลขึ้น Sheet" หลังแก้ไข และให้เครื่องอื่นกด
> "ดึงข้อมูลจาก Sheet" เพื่ออัปเดต ข้อมูลจะถูกเก็บเป็น JSON ก้อนเดียวในชีตชื่อ `data` เซลล์ `A1`

## หมายเหตุ

- ถ้าแก้โค้ด `Code.gs` แล้ว ต้อง **Deploy ใหม่** (Manage deployments → แก้ไข version) ลิงก์ `/exec` จะเหมือนเดิม
- ข้อมูลรูปภาพถูกเก็บเป็น base64 ซึ่งกินพื้นที่มาก หากรูปเยอะอาจชนขนาดสูงสุดของเซลล์ (~50,000 ตัวอักษร)
  กรณีใช้งานจริงจังแนะนำเก็บรูปแยก (เช่น Google Drive) แล้วเก็บเฉพาะลิงก์
